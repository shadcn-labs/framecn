**Video components built with Editframe** for Next.js applications. Create, edit, and render videos programmatically.

## What You Get

- 🎬 **Editframe integration** - Build video compositions with HTML, React components
- 📹 **Video rendering** - Export to MP4 via cloud rendering
- 📚 **Documentation site** - Powered by Fumadocs
- 🤖 **[Agent ready](https://www.mintlify.com/score/framecn)** - Ships with `llms.txt`, `llms-full.txt`, API catalog routes, and agent skill discovery endpoints
- 🔊 **[Web audio feedback](https://audio.raphaelsalaja.com/)** - Built-in interaction sounds powered by `@web-kits/audio`
- 📳 **[Web haptics](https://haptics.lochie.me/)** - Optional haptic feedback for supported devices via `web-haptics`
- ✨ **[Motion animations](https://motion.dev/)** - Uses `motion` for animated UI states and transitions
- 🎯 **[Animated icons](https://lucide-animated.com/)** - Includes reusable animated icons for buttons, sharing, and navigation
- 🔄 **[View transitions](https://nextjs.org/docs/app/api-reference/config/next-config-js/viewTransition)** - Uses Next.js view transitions for smoother page changes
- 🚀 **Deploy ready** - Deploy anywhere

## Tech Stack

- `Next.js 16` + App Router
- `React 19`
- `Tailwind CSS 4`
- `Fumadocs`
- `Editframe` - Video composition and rendering
- `motion`
- `shiki` + `rehype-pretty-code`

## How It Works

1. Install the `@editframe/elements` package
2. Import video components into your Next.js app
3. Create compositions with videos, images, text, and effects

That's it. Deploy your app and start building video experiences:

```bash
npm install @editframe/elements
```
