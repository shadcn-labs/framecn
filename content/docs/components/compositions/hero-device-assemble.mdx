## API Reference

### HeroDeviceAssemble

| Prop          | Type         | Default   |
| ------------- | ------------ | --------- |
| `device`      | `"laptop" \` | `"phone"` |
| `showShimmer` | `boolean`    | `true`    |

## Notes

<Callout title="Two-phase device assembly">
  Devices spring into place from floating layers, then the screen wakes with a
  shimmer effect. The device prop selects laptop or phone.
</Callout>
