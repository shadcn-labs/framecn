## Usage

```tsx
import { PerspectiveMarquee } from "@/components/framecn/perspective-marquee";
```

```tsx
<PerspectiveMarquee text="text" perspective={0} speed={0} />
```

## API Reference

### PerspectiveMarquee

| Prop          | Type     | Default |
| ------------- | -------- | ------- |
| `text`        | `string` | `-`     |
| `perspective` | `number` | `500`   |
| `speed`       | `number` | `50`    |

## Notes

<Callout title="3D perspective creates depth">
  The marquee uses CSS perspective and transform3D for the 3D effect. Text fades
  in opacity as it moves deeper into the scene.
</Callout>
