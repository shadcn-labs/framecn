## API Reference

### MarkerHighlight

| Prop          | Type      | Default     |
| ------------- | --------- | ----------- |
| `text`        | `string`  | `-`         |
| `markerColor` | `string`  | `"#fbbf24"` |
| `invertText`  | `boolean` | `false`     |
