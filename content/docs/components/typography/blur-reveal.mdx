## Usage

```tsx

```

```tsx
<BlurReveal text="text" blur={0} fontSize={0} fontWeight={0} color="color" />
```

## API Reference

### BlurReveal

| Prop         | Type     | Default     |
| ------------ | -------- | ----------- |
| `text`       | `string` | `-`         |
| `blur`       | `number` | `10`        |
| `fontSize`   | `number` | `48`        |
| `fontWeight` | `number` | `600`       |
| `color`      | `string` | `"#171717"` |

## Notes

<Callout title="Blur uses backdrop-filter">
  The blur effect uses CSS backdrop-filter which may not be supported in all
  browsers. Consider providing a fallback for older browsers.
</Callout>
