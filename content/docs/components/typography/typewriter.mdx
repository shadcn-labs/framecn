
## API Reference

### Typewriter

| Prop       | Type      | Default |
| ---------- | --------- | ------- |
| `text`     | `string`  | `-`     |
| `speed`    | `number`  | `0.08`  |
| `cursor`   | `boolean` | `true`  |
| `fontSize` | `number`  | `48`    |
