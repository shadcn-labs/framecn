## Usage

```tsx

```

```tsx
<StrikethroughReplace fromText="fromText" toText="toText" duration={0} />
```

## API Reference

### StrikethroughReplace

| Prop       | Type     | Default |
| ---------- | -------- | ------- |
| `fromText` | `string` | `-`     |
| `toText`   | `string` | `-`     |
| `duration` | `number` | `0.5`   |

## Notes

<Callout title="Diagonal line cuts through text">
  The strikethrough line cuts diagonally from top-left to bottom-right. The
  animation syncs the old text removal with the new text reveal.
</Callout>
