## API Reference

### CaptionNeonGlow

| Prop               | Type               | Default      |
| ------------------ | ------------------ | ------------ |
| `words`            | `CaptionWord[]`    | (demo words) |
| `glowColor`        | `string`           | `"#00fff0"`  |
| `fontSize`         | `number`           | `80`         |
| `fontWeight`       | `number \| string` | `900`        |
| `wordsPerGroup`    | `number`           | `6`          |
| `speed`            | `number`           | `1`          |
| `fps`              | `number`           | `30`         |
| `durationInFrames` | `number`           | `180`        |
| `width`            | `number`           | `1280`       |
| `height`           | `number`           | `720`        |
| `className`        | `string`           | `-`          |

## Notes

<Callout title="Word timing is in seconds">
  The `start` and `end` fields on each `CaptionWord` are in seconds.
</Callout>
