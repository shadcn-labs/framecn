## API Reference

### CaptionWeightShift

| Prop               | Type            | Default                    |
| ------------------ | --------------- | -------------------------- |
| `words`            | `CaptionWord[]` | (demo words)               |
| `color`            | `string`        | `"#ffffff"`                |
| `activeColor`      | `string`        | `"#ffffff"`                |
| `dimColor`         | `string`        | `"rgba(255,255,255,0.35)"` |
| `fontSize`         | `number`        | `72`                       |
| `speed`            | `number`        | `1`                        |
| `fps`              | `number`        | `30`                       |
| `durationInFrames` | `number`        | `180`                      |
| `width`            | `number`        | `1280`                     |
| `height`           | `number`        | `720`                      |
| `className`        | `string`        | `-`                        |

## Notes

<Callout title="Word timing is in seconds">
  The `start` and `end` fields on each `CaptionWord` are in seconds.
</Callout>
