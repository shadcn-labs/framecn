## API Reference

### CaptionBlendDifference

| Prop               | Type               | Default             |
| ------------------ | ------------------ | ------------------- |
| `words`            | `CaptionWord[]`    | (demo words)        |
| `wordsPerGroup`    | `number`           | `3`                 |
| `color`            | `string`           | `"#ffffff"`         |
| `fontSize`         | `number`           | `72`                |
| `fontWeight`       | `number \| string` | `800`               |
| `background`       | `string`           | (colorful gradient) |
| `speed`            | `number`           | `1`                 |
| `fps`              | `number`           | `30`                |
| `durationInFrames` | `number`           | `180`               |
| `width`            | `number`           | `1280`              |
| `height`           | `number`           | `720`               |
| `className`        | `string`           | `-`                 |

## Notes

  The `start` and `end` fields on each `CaptionWord` are in seconds.
