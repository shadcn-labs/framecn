## API Reference

### CaptionClipWipe

| Prop               | Type               | Default               |
| ------------------ | ------------------ | --------------------- |
| `words`            | `CaptionWord[]`    | (demo words)          |
| `keywords`         | `string[]`         | `["single", "code."]` |
| `color`            | `string`           | `"#ffffff"`           |
| `keywordColor`     | `string`           | `"#FFD700"`           |
| `fontSize`         | `number`           | `72`                  |
| `fontWeight`       | `number \| string` | `700`                 |
| `background`       | `string`           | `"#0a0a0a"`           |
| `speed`            | `number`           | `1`                   |
| `fps`              | `number`           | `30`                  |
| `durationInFrames` | `number`           | `180`                 |
| `width`            | `number`           | `1280`                |
| `height`           | `number`           | `720`                 |
| `className`        | `string`           | `-`                   |
