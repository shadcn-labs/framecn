
```tsx

```

## API Reference

### CaptionEditorialEmphasis

| Prop               | Type               | Default               |
| ------------------ | ------------------ | --------------------- |
| `words`            | `CaptionWord[]`    | (demo words)          |
| `emphasisWords`    | `string[]`         | `["single", "code."]` |
| `color`            | `string`           | `"#f5f0d0"`           |
| `fontSize`         | `number`           | `56`                  |
| `fontWeight`       | `number \| string` | `"400"`               |
| `emphasisScale`    | `number`           | `2`                   |
| `speed`            | `number`           | `1`                   |
| `fps`              | `number`           | `30`                  |
| `durationInFrames` | `number`           | `180`                 |
| `width`            | `number`           | `1280`                |
| `height`           | `number`           | `720`                 |
| `className`        | `string`           | `-`                   |

## Notes

<Callout title="Word timing is in seconds">
  The `start` and `end` fields on each `CaptionWord` are in seconds.
</Callout>
