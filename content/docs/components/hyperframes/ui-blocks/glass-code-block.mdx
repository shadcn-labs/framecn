## Usage

Add the composition to your host HTML file:

```html

  data-composition-id="glass-code-block"
  data-composition-src="compositions/glass-code-block.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```
