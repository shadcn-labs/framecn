## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="chromatic-aberration-wipe"
  data-composition-src="compositions/components/chromatic-aberration-wipe.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="RGB offset on peak frames">
  The RGB channels split and offset during the peak of the transition, creating
  a glitch effect. The intensity controls the offset amount.
</Callout>
