## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="directional-wipe"
  data-composition-src="compositions/components/directional-wipe.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```
