## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="spatial-push"
  data-composition-src="compositions/components/spatial-push.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="3D perspective push">
  The old scene is pushed back in 3D space while the next scene slides in from
  the chosen `direction`. Use `transitionDuration` to control how long the push
  phase takes.
</Callout>
