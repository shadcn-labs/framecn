## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="zoom-through-transition"
  data-composition-src="compositions/components/zoom-through-transition.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

<Callout title="Scale-through effect">
  The transition scales through a given element to create an aggressive zoom
  effect. Higher scale values create more dramatic transitions.
</Callout>
