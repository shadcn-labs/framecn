## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="frosted-glass-wipe"
  data-composition-src="compositions/components/frosted-glass-wipe.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```
