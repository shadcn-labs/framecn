## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="ecosystem-constellation"
  data-composition-src="compositions/ecosystem-constellation.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```
