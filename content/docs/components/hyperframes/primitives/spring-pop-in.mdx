## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="spring-pop-in"
  data-composition-src="compositions/components/spring-pop-in.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Spring-based overshoot">
  The animation uses a spring function with configurable overshoot. Tune
  `stiffness`, `damping`, and `mass` to shape how far it overshoots before
  settling.
</Callout>
