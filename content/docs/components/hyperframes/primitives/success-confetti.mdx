## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="success-confetti"
  data-composition-src="compositions/components/success-confetti.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Deterministic particle physics">
  The confetti uses deterministic physics with seeded random values. Same inputs
  always produce the same particle patterns.
</Callout>

<Callout title="Performance scales with particle count" variant="warning">
  Higher particle counts may impact performance. Keep particleCount under 100
  for smooth animations.
</Callout>
