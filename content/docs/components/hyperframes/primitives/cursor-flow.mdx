Add the composition to your host HTML file:

```html

  data-composition-id="cursor-flow"
  data-composition-src="compositions/components/cursor-flow.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
```

## Notes

<Callout title="Bezier arcs between waypoints">
  The cursor follows per-segment cubic Bezier curves with perpendicular control
  points. The arc creates natural, realistic movement.
</Callout>

<Callout title="No d3-shape dependency" variant="warning">
  Curves are computed inline to avoid pulling in external dependencies. Path
  math is handled internally.
</Callout>
