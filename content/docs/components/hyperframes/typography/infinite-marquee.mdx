## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="infinite-marquee"
  data-composition-src="compositions/components/infinite-marquee.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Uses CSS transform for performance">
  The marquee uses CSS transforms for smooth scrolling. The text is duplicated
  to create the infinite loop effect.
</Callout>

<Callout title="May cause layout shift on resize" variant="destructive">
  The marquee calculates width based on initial render. If the container
  resizes, you may need to force a re-render.
</Callout>
