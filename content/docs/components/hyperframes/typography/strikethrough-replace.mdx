## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="strikethrough-replace"
  data-composition-src="compositions/components/strikethrough-replace.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

<Callout title="Diagonal line cuts through text">
  The strikethrough line cuts diagonally from top-left to bottom-right. The
  animation syncs the old text removal with the new text reveal.
