## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="perspective-marquee"
  data-composition-src="compositions/components/perspective-marquee.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="3D perspective creates depth">
  The marquee uses CSS perspective and transform3D for the 3D effect. Text fades
  in opacity as it moves deeper into the scene.
</Callout>
