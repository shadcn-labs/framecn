### RGBGlitchText

| Prop               | Type     | Default     |
| ------------------ | -------- | ----------- |
| `text`             | `string` | `-`         |

## Notes

<Callout title="RGB channels offset with jitter">
  The text renders three offset channels (red, green, blue) with randomized
  horizontal jitter. The intensity controls the offset amount.
</Callout>
