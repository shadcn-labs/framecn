## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="shimmer-sweep"
  data-composition-src="compositions/components/shimmer-sweep.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Gradient direction is left-to-right">
  The shimmer effect sweeps from left to right. You can reverse this by swapping
  the gradient colors or using a custom gradient direction.
</Callout>
