## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="staggered-fade-up"
  data-composition-src="compositions/components/staggered-fade-up.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Stagger delay controls cascade timing">
  The `staggerDelay` prop controls the delay between each word's animation.
  Lower values create a tighter cascade, higher values create a more dramatic
  delay.
</Callout>
