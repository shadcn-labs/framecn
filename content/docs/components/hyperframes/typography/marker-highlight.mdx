## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="marker-highlight"
  data-composition-src="compositions/components/marker-highlight.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Marker uses absolute positioning">
  The marker bar is positioned absolutely behind the text. Ensure the parent has
  `position: relative` for correct placement.
</Callout>

<Callout title="Text inversion requires contrast" variant="destructive">
  When `invertText` is true, the text color inverts. Ensure your marker color
  has sufficient contrast with the background.
</Callout>
