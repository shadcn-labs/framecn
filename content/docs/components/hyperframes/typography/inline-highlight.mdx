## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="inline-highlight"
  data-composition-src="compositions/components/inline-highlight.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

  Pass an array of word indices to `highlightWords` to animate multiple words.
  The animation sequences through each word in the array.
