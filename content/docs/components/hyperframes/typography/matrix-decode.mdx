## Usage

Add the composition to your host HTML file:

```html
<div
  data-composition-id="matrix-decode"
  data-composition-src="compositions/components/matrix-decode.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
></div>
```

## Notes

<Callout title="Random characters resolve to target">
  The decode effect uses random characters from a Matrix-style character set
  that gradually resolve to the target text. Longer texts take more time to
  fully resolve.
</Callout>
