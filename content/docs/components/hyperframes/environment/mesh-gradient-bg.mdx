Add the composition to your host HTML file:

```html

  data-composition-id="mesh-gradient-bg"
  data-composition-src="compositions/components/mesh-gradient-bg.html"
  data-start="0"
  data-duration="8"
  data-track-index="1"
  data-width="1920"
  data-height="1080"
```
