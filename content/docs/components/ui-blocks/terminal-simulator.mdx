## Usage

```tsx
import { TerminalSimulator } from "@/components/framecn/terminal-simulator";
```

```tsx
<TerminalSimulator commands="commands" typingSpeed={0} showPrompt={true} />
```

## API Reference

### TerminalSimulator

| Prop          | Type       | Default |
| ------------- | ---------- | ------- |
| `commands`    | `string[]` | `-`     |
| `typingSpeed` | `number`   | `0.05`  |
| `showPrompt`  | `boolean`  | `true`  |

## Notes

<Callout title="CLI typing simulation">
  Commands type out character-by-character with chunked log output. The terminal
  auto-scrolls as new content appears.
</Callout>
