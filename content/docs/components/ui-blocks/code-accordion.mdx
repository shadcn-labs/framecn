
## API Reference

### CodeAccordion

| Prop        | Type     | Default        |
| ----------- | -------- | -------------- |
| `code`      | `string` | `-`            |
| `threshold` | `number` | `10`           |
| `language`  | `string` | `"typescript"` |
