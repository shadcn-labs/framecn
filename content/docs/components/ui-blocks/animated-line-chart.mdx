## API Reference

### AnimatedLineChart

| Prop     | Type                              | Default     |
| -------- | --------------------------------- | ----------- |

| `width`  | `number`                          | `400`       |
| `height` | `number`                          | `200`       |
| `color`  | `string`                          | `"#3b82f6"` |
