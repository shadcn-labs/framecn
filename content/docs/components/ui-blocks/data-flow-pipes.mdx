## API Reference

### DataFlowPipes

| Prop               | Type             | Default         |

| `nodes`            | `DataFlowNode[]` | `DEFAULT_NODES` |

| `pipeColor`        | `string`         | `"#1f1f23"`     |
| `pulseColor`       | `string`         | `"#22d3ee"`     |
| `pulseLength`      | `number`         | `60`            |
| `pulseDuration`    | `number`         | `36`            |
| `background`       | `string`         | `"#050505"`     |
| `nodeColor`        | `string`         | `"#0a0a0a"`     |

| `speed`            | `number`         | `1`             |

| `className`        | `string`         | `-`             |
