## Usage

```tsx
import { StaggeredBentoGrid } from "@/components/framecn/staggered-bento-grid";
```

```tsx
<StaggeredBentoGrid columns={0} gap={0} staggerDelay={0} />
```

## API Reference

### StaggeredBentoGrid

| Prop           | Type     | Default |
| -------------- | -------- | ------- |
| `columns`      | `number` | `3`     |
| `gap`          | `number` | `16`    |
| `staggerDelay` | `number` | `0.1`   |

## Notes

<Callout title="Per-cell cascade animation">
  Cards cascade in with staggered timing. Each cell has its own animation
  sequence for a dynamic entrance effect.
</Callout>
