## API Reference

### StaggeredFadeUp

| Prop               | Type     | Default     |
| ------------------ | -------- | ----------- |
| `text`             | `string` | `-`         |
| `staggerDelay`     | `number` | `4`         |
| `distance`         | `number` | `20`        |
| `fontSize`         | `number` | `72`        |
| `color`            | `string` | `"#171717"` |
| `fontWeight`       | `number` | `600`       |
| `speed`            | `number` | `1`         |
| `fps`              | `number` | `30`        |
| `durationInFrames` | `number` | `90`        |
| `className`        | `string` | `-`         |

## Notes

<Callout title="Stagger delay controls cascade timing">
  The `staggerDelay` prop controls the delay between each word's animation.
  Lower values create a tighter cascade, higher values create a more dramatic
  delay.
</Callout>
