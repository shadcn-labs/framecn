## API Reference

### FrostedGlassWipe

| Prop                 | Type        | Default   |
| -------------------- | ----------- | --------- |
| `from`               | `ReactNode` | `-`       |
| `to`                 | `ReactNode` | `-`       |
| `transitionStart`    | `number`    | `36`      |
| `transitionDuration` | `number`    | `30`      |
| `glassBlur`          | `number`    | `24`      |
| `speed`              | `number`    | `1`       |
| `fps`                | `number`    | `30`      |

## Notes

<Callout title="Uses backdrop-filter">
  The frosted glass effect uses CSS backdrop-filter for the blur. Browser
  support is good but not universal.
</Callout>
