```tsx

```

```tsx
<DynamicGrid pattern="value" spacing={0} color="color" animate={true} />
```

## API Reference

### DynamicGrid

| Prop      | Type       | Default     |
| --------- | ---------- | ----------- |
| `pattern` | `"grid" \` | `"dots"`    |
| `spacing` | `number`   | `40`        |
| `color`   | `string`   | `"#e5e7eb"` |
| `animate` | `boolean`  | `true`      |
