
```tsx

```

## API Reference

### SpatialPush

| Prop        | Type       | Default   |
| ----------- | ---------- | --------- |
| `direction` | `"left" \` | `"right"` |
| `depth`     | `number`   | `200`     |

## Notes

<Callout title="3D perspective push">
  The old scene is pushed back in 3D space using transform3D. The depth prop
  controls how far back it gets pushed.
</Callout>
