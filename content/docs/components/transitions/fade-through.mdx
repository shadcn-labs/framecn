## API Reference

### FadeThrough

| Prop               | Type     | Default         |
| ------------------ | -------- | --------------- |
| `fromText`         | `string` | `"Hello world"` |
| `toText`           | `string` | `"New world"`   |
| `fontSize`         | `number` | `72`            |
| `color`            | `string` | `"#171717"`     |
| `fontWeight`       | `number` | `600`           |
| `speed`            | `number` | `1`             |
| `fps`              | `number` | `30`            |
| `durationInFrames` | `number` | `90`            |
| `className`        | `string` | `-`             |

## Notes

<Callout title="Overlapping fade timing">
  The enter animation begins one frame after the exit animation finishes,
  creating a smooth handoff. Adjust `speed` to control the overall pace of the
  transition.
</Callout>
