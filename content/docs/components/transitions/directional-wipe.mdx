
```tsx

```

## API Reference

### DirectionalWipe

| Prop        | Type       | Default     |
| ----------- | ---------- | ----------- |
| `direction` | `"left" \` | `"right" \` |
| `duration`  | `number`   | `0.5`       |

## Notes

<Callout title="Push-style transition">
  One scene slides in while pushing the other out in the specified direction.
  The duration controls the transition speed.
</Callout>
